Avatar

Avatars are used to show user's profile picture on an application and are available in different sizes.

  • avatar-x-large   - font-size: 4rem
  • avatar-large      - font-size: 3.2rem
  • avatar-regular  - font-size: 2.4rem
  • avatar-small     - font-size: 1.6rem
  • avatar-x-small   - font-size: 10px

Note :

 Make sure to use 3 classes for proper results - avatar, avatar-regular (or any other size class) on div tag and  avatar-img on image tag
