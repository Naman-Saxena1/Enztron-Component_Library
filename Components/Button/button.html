Button

Buttons are used to take some action or do any particular task.
Enztron has 4 types of button components -
1) Primary Buttons
2) Ouline Buttons
3) Link Buttons
4) Icon Buttons
5) Floating Action Button





Solid Buttons



                        
<button class="solid-primary-btn">Primary Button</button>
<button class="solid-secondary-btn">Secondary Button</button>
<button class="solid-error-btn">Error Button</button>
<button class="solid-success-btn">Success Button</button>
<button class="solid-warning-btn">Warning Button</button>


Outline Buttons



                        
<button class="outline-primary-btn">Primary Button</button>
<button class="outline-secondary-btn">Secondary Button</button>
<button class="outline-error-btn">Error Button</button>
<button class="outline-success-btn">Success Button</button>
<button class="outline-warning-btn">Warning Button</button>


Link Buttons



                        
<button class="solid-primary-btn link-btn"><a href="...">Primary Solid Link Button</a></button>
<button class="outline-primary-btn link-btn"><a href="...">Primary Outline Link Button</a></button>
<button class="solid-secondary-btn link-btn"><a href="...">Secondary Solid Link Button</a></button>
<button class="outline-secondary-btn link-btn"><a href="...">Secondary Outline Link Button</a></button>


Icon Buttons



                        
<button class="icon-btn"><i class="fa fa-heart-o" aria-hidden="true"></i></button>
<button class="icon-btn"><i class="fa fa-thumbs-o-up" aria-hidden="true"></i></button>
<button class="icon-btn"><i class="fa fa-bars" aria-hidden="true"></i></button>
<button class="icon-btn"><i class="fa fa-share-alt" aria-hidden="true"></i></button>
<button class="icon-btn"><i class="fa fa-ellipsis-v" aria-hidden="true"></i></button>


Floating Action Buttons

It is the button which floats with the UI represents the primary action in an application
Example : It is the + button on this page that is floating with scroll 



                        
<button class="icon-btn floating-action-btn"><i class="fa fa-plus" aria-hidden="true"></i></button>
