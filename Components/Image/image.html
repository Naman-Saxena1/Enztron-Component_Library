Image

Responsive images are used to display appropriate image according to screen dimensions, density, resolution & art direction (landscape/potrait mode).





Responsive Image



Responsive Image Demo



Note :
1) For this demo while in Chrome Dev Tools responsive mode, I have set DPR : 1 at top-center bar settings. If you use other DPR values, browser will pick best possible image according to it.
2) For demo purpose I used different images with different dimensions so that difference is noticable but in practical we would be using same image with different dimensions. 



                        
<img
    src="https://picsum.photos/id/338/100"
    srcset="https://picsum.photos/id/338/400 410w,
            https://picsum.photos/id/650/600 600w,
            https://picsum.photos/id/450/800 800w"
    sizes=" (min-width: 1240px) 700px, 
            (min-width: 960px)  600px, 400px"
/>


Square Images

Above image is original image and below is same image with class square-image
Note : It shrinks image to fit into square because object-fit: fill which you can override according to your needs. 



                        
<img src="..." 
     alt="mountain-image"/>
<img class="square-image"
     src="..." 
     alt="mountain-image"/>
