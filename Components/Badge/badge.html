Badge

Badges are used to show extra information to user like online/offline status, number of unread notifications, number of items in a cart , etc.

    <i class="fa fa-shopping-cart fa-x" aria-hidden="true" ></i>
    <span class="count-badge-x">5</span>
</div>

Note :

1) Make sure to use 2 classes for proper results -
   icon-count-badge on container div and count-badge-5x (or any other badge size) on badge span

2) Make sure to use badge size class that matches Font Awesome icon size
    Example - count-badge-2x with fa-2x

<div class="avatar avatar-x-large">
    <img class="avatar-img" src="..." alt="avatar"/>
    <span class="status-badge-5x status-online"></span>
</div>

<div class="avatar avatar-large">
    <img class="avatar-img" src="..." alt="avatar"/>
    <span class="status-badge-4x status-offline"></span>
</div>

<div class="avatar avatar-regular">
    <img class="avatar-img" src="..." alt="avatar"/>
    <span class="status-badge-3x status-busy"></span>
</div>

<div class="avatar avatar-small">
    <img class="avatar-img" src="..." alt="avatar"/>
    <span class="status-badge-2x status-away"></span>
</div>

<div class="avatar avatar-x-small">
    <img class="avatar-img" src="..." alt="avatar"/>
    <span class="status-badge-x status-away"></span>
</div>

Note :

Make sure to use 2 classes for proper results -
    1) status-badge-3x (or of any other size according to profile size) on status span
       There are 5 sizes of status class - status-badge-5x, status-badge-4x, status-badge-3x, status-badge-2x and        status-badge-x

    2) status-online (or any other status type)
       There are 4 types of status class - status-online, status-offline, status-busy and status-away
